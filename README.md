

   
# 功能与目的：

<br>

本项目核心功能：

 - 生成 homebrew + sparkle 所需的 clash 分流规则
   
 - 已防止 因为GFW 而影响 Mac APP 升级安装过程 （如：brew upgrade \ brew install \ sparkle ）

# 使用方法：

<br>

- 执行：
    ```
    ./usercmd_MacAppUpgrade_to_clashruleset
    ```


- 生成的规则 在 当前目录 ：
    ```
    ./MacAppUpgrade_Domain.yaml
    ./MacAppUpgrade_No_Resolve.yaml
    ./MacAppUpgrade.yaml
     ```
