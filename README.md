 - homebrew + sparkle 所需的 clash 分流规则

使用方法：
--------------------------------

- 执行：
    ```
    ./usercmd_MacAppUpgrade_to_clashruleset
    ```


- 生成的规则 在 当前目录 ：
    ```
    ./MacAppUpgrade_Domain.yaml
    ./MacAppUpgrade_No_Resolve.yaml
    ./MacAppUpgrade.yaml
     ```
